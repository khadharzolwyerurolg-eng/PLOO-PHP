## O que são Modificadores de Acesso?

São palavras-chave que controlam quem pode ver ou alterar as informações dentro de uma classe. Eles servem para garantir o encapsulamento, que é a proteção dos dados do objeto.

## Os Três Modificadores Principais

    ## Público (public):
        Visibilidade: Total.
        Acesso: Qualquer outra classe do sistema pode aceder e modificar. 

    ## Privado (private)
        Visibilidade: Restrita.
        Acesso: Apenas a própria classe onde foi criado pode aceder. Classes externas não têm acesso direto. 
        
    Protegido (protected)
        Visibilidade: Intermédia.
        Acesso: Apenas a própria classe e as suas subclasses (herança) podem aceder.
