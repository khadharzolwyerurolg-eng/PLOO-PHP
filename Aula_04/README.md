## Método construtor:
O Método Construtor é um bloco de código especial executado automaticamente assim que uma nova instância (objeto) de uma classe é criada. A sua principal função é garantir que o objeto nasça com os seus atributos inicializados corretamente.

📌 Características Principais:
## Execução automática: 
    É chamado apenas uma vez, no momento do new.

## Mesmo nome: 
    Em muitas linguagens (como Java, C# e C++), tem exatamente o mesmo nome da classe.

## Sem tipo de retorno: 
    Não retorna nenhum valor, nem mesmo void.

## Inicialização: 
    Serve para definir os valores iniciais dos atributos do objeto.
