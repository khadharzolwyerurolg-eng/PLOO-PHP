## O que são classes e objectos?
As classes e objectos são abstrações ou características das entidades no mundo real.
Exemplo:
    * Carro;
    * Pessoa;
    * Animal;

## Elementos da classe:
As classes são compostas basicamente por:
    * Um nome;
    * Atributos;
    * Métodos;

    Exemplo:
        Classe Carro:
            Atributo(Características):
                #--> Cor;
                #--> Número de portas;
            
            Métodos(Funções);
                #--> Buzinar;
                #--> Acelerar;

NOTA: ->: Usa-se para acessar atributos ou métodos.
